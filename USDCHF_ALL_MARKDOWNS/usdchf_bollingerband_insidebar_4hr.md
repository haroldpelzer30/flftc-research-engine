# Preliminary Conclusion

This is the first Bollinger Band study that materially exceeded every previous USDCHF BB result.

However:

* Only 66 bearish signals were produced.
* The sample size is extremely small.
* The result is statistically fragile until replicated elsewhere.

---

# Research Takeaway

The USDCHF Inside Bar study presents the strongest Bollinger Band statistic observed so far.

However, the result comes with a major caveat:

> **Extremely selective filters can create impressive headline statistics while leaving very little evidence about whether the edge is robust.**

The result is interesting, but the sample size is too small to confidently conclude that Inside Bars create a durable USDCHF trading edge.
