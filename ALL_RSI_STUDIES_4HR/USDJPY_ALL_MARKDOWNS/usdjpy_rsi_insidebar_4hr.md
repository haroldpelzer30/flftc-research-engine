# Research Interpretation

The Inside Bar filter produced the strongest USDJPY result observed so far.

However:

* Opportunity count was dramatically reduced.
* The improvement was concentrated in one holding period.
* Performance outside that horizon remained close to random.

This makes it difficult to conclude that Inside Bars create a robust edge in USDJPY.

Current USDJPY Ranking:

1. RSI + Inside Bar — 54.14%
