# Key Findings

### 1. Strongest NZDUSD Result So Far

The bullish Inside Bar setup produced:

* 57.30%
* 1-Candle Hold
* 281 Signals

This is the highest win rate observed in any NZDUSD study so far.

---

### 2. Significant Reduction in Opportunities

Signal count fell from:

* 3,343 RSI Alone signals

to:

* 553 Inside Bar signals

representing an approximately 83% reduction in opportunities.

---

### 3. Mixed Performance Across Holding Periods

The bullish side showed substantial variability:

| Horizon   | Win Rate |
| --------- | -------- |
| 1 Candle  | 57.30%   |
| 3 Candles | 49.11%   |
| 6 Candles | 54.80%   |

The edge did not remain consistent across holding periods.

---

### 4. Bearish Results Were More Moderate

Bearish performance remained respectable:

* 54.41%
* 52.57%

before deteriorating to:

* 49.26%

at the 6-candle horizon.

---

# Research Interpretation

This is the first NZDUSD filter that produced a result comparable to USDCHF-level performance.

However, several caveats remain:

* Signal count was reduced substantially.
* The strongest result occurred only at a specific holding period.
* Performance varied significantly across horizons.

This makes it difficult to conclude that the Inside Bar filter consistently improves NZDUSD RSI signals.

---

# Preliminary Conclusion

NZDUSD RSI + Inside Bar currently ranks as the strongest NZDUSD study.

Current ranking:

1. RSI + Inside Bar — 57.30%
2. RSI + Double Bottom / Top — 54.70%
3. RSI + Daily Structure — 53.31%
4. RSI Alone — 52.86%

The result is encouraging but should be interpreted cautiously due to the large reduction in opportunity count and inconsistent performance across holding periods.
